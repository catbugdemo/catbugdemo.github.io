2. 安装,使用git

安装地址

  • 设置 user.name 和 user.email
  • 生成ssh密钥文件 找到 ~/.ssh 文件中的 id_rsa.pub密钥,将内容全部复制

打开GitHub->Settings->SSH and GPG keys ,新建new SSH key,然后将刚刚复制id_rsa.pub的内容全部粘贴进去，然后点击Add SSH key。

输入

当出现 Hi,你的GitHub名称 时成功
