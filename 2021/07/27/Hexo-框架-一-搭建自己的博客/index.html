hexo 基本操作

  • 本地启动项目，s表示server

  • 创建不可
  • layout : 文章布局，可选（post page draft），默认使用 _config.yml 中的 default_layout 参数代替（post）。
  • title : 文章标题。

  • 生成静态文件，g表示generate

  • 部署Hexo网站，d表示deploy
  • 清除缓存文件 (db.json) 和已生成的静态文件 (public)
